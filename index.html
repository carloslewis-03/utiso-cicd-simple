UTISO CI/CD Demo

A simple static site published by a secure GitHub Actions pipeline.

Deployed via GitHub Actions

This page is intentionally simple. The point of the repository is the CI/CD pipeline that publishes it, not the page itself.

What the pipeline does

  • Scans for secrets in the code before anything ships (Gitleaks).
  • Deploys to GitHub Pages — but only if the secret scan passes.

Why it matters

Every change is checked automatically and published the same way every time, with a security gate in front of the deploy. No manual steps, no forgotten checks.
